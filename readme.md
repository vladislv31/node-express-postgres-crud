## CRUD API

### Usage

- **Clone repository**

- **Create config for database:**

File: config/db.config.js

```
module.exports = {
    user: '',
    password: '',
    host: '',
    db: ''    
}
```

- Import database **db.sql**

- Start server:
    - ```npm run start``` - for production
    - ```npm run dev``` - for development
